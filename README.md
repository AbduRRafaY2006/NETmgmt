# NUST Entry Test Management System

## Overview

The **NUST Entry Test Management System** is a simple academic web application developed for a **Database Systems** course. It models entry test **form filling and scheduling** using a relational database.

> This is **not an official NUST system**. It is built **only for educational purposes**.

## Tech Stack

* **Backend:** Node.js, Express.js
* **Frontend:** HTML, CSS, JavaScript
* **Database:** MySQL

## Features

* Candidate registration
* Entry test form submission
* Test scheduling
* Applicant detail viewing
* MySQL-based data storage

## Academic Note

* Project name and parts of the business logic are **inspired by real NUST entry test procedures**.
* Database design, SQL queries, and implementation were done **independently for academic learning**.

## Setup (Local)

1. Import the MySQL schema
2. Configure DB credentials
3. Run:

   ```bash
   npm install
   node server.js
   ```
